# Lismore Investigation Framework
## Detective-Prosecutor Autonomous Investigation System

### Quick Start
1. Place documents in documents\raw\batch_001\
2. Add API key to config\.env
3. Run: python src\main.py
4. Review findings after each spiral in outputs\spiral_reports\

### Investigation Process
- **Spiral 1**: Complete understanding (Detective)
- **Spiral 2**: Anomaly detection (Detective)  
- **Spiral 3**: Theory building (Prosecutor)
- **Spiral 4**: Kill shots (Prosecutor)

### For Paralegals
All findings include:
- Full document references
- Evidence trails
- Importance rankings
- Source traceability
